# Personal Homepage Template

这是一个可直接部署到 GitHub Pages 的静态个人主页模板，风格接近目标站点。

## 使用

1. 修改 `index.html` 里的姓名、简介、学校、项目、链接。
2. 修改 `style.css` 里的颜色、字体或间距。
3. 把头像放到 `assets/files/avatar/avatar.jpg`。
4. 如果需要项目配图，把占位块改成 `<img>` 或 `<video>`。
5. 推送到名为 `yourname.github.io` 的仓库，开启 GitHub Pages。

## 目录建议

```text
assets/
└─ files/
   ├─ avatar/
   ├─ work/
   ├─ icon/
   └─ institute/
```

## 发布

仓库名使用 `你的 GitHub 用户名.github.io`，推送后在仓库设置里开启 Pages 即可。
